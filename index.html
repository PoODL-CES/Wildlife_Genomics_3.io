Welcome to Wildlife Genomics 3 workshop

Dive into the world of wild genomics! This hands-on workshop is a way to make you accustomed to the cutting edge genomics tools and population genetics, all under one place and tailored for wildlife research and conservation. From decoding fundamental genetic principles to analyzing Illumina-generated Next-generation sequence (NGS) data analysis. Learn how to apply genomic insights to wildlife censuses, combat trafficking, and inform population management. With a strong focus on practical data analysis, this course also equips you to adapt the workflow for RAD-seq or amplicon-seq, if you've got a reference genome, you're good to go!.

Workshop Objectives

By the end of this workshop, you will be able to:

  • Understand and process raw sequencing data (FASTQ format)
  • Perform quality control and preprocessing
  • Map reads to a reference genome
  • Identify variants (SNPs and indels)
  • Use filtered variants for basic population genetic analysis (PCA, Admixture, heterozygosity)

Full version of the workshop is available at: PoODL-CES GitHub
